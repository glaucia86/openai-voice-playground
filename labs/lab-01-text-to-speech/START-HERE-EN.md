# Start here — Lab 01

This branch is the executable starting point for the Text-to-Speech workshop.

1. Open a terminal in this directory.
2. Run `npm ci`.
3. Copy `.env.example` to `.env.local` and fill in `OPENAI_API_KEY` only when the tutorial asks you to.
4. Run `npm run dev` and open `http://localhost:3000`.
5. Follow the [English tutorial](tutorial/tutorial-en.md), starting with **path B — recommended starter**.

The base must pass `npm run check` without calling OpenAI. Inspect checkpoints with `git show` or `git diff`; do not switch branches over unfinished work without creating your own branch first.
