# Comece aqui — Lab 01

Esta branch é o ponto de partida executável do workshop de Text to Speech.

1. Abra o terminal nesta pasta.
2. Execute `npm ci`.
3. Copie `.env.example` para `.env.local` e preencha `OPENAI_API_KEY` somente quando o tutorial solicitar.
4. Execute `npm run dev` e abra `http://localhost:3000`.
5. Siga o [tutorial em português](tutorial/tutorial.md), começando pelo caminho **B — starter recomendado**.

A base deve passar em `npm run check` sem realizar chamadas à OpenAI. Consulte os checkpoints com `git show` ou `git diff`; não troque de branch em cima do seu trabalho sem antes criar seu próprio branch.
